# טריוויה יומית (MVP v1)

אפליקציית PWA פשוטה של שאלה אחת ביום לכולם.

## מה יש בפנים

- שאלה יומית אחת מתוך מאגר (כרגע 7 שאלות)
- 4 תשובות אמריקאיות
- משוב מיידי + הסבר קצר
- שמירת ניקוד ורצף יומי ב־`localStorage`
- כפתור שיתוף תוצאה (בלי לחשוף תשובה)
- תמיכה ב־PWA להתקנה במובייל
- מצב כהה/בהיר

## הרצה מקומית

1. התקן Node.js (מומלץ 20+)
2. התקן תלויות:

```bash
npm install
```

3. הרץ:

```bash
npm run dev
```

4. פתח:
[http://localhost:3000](http://localhost:3000)

## קבצים חשובים

- `src/app/page.tsx` - המסך הראשי ולוגיקת המשחק
- `src/data/questions.json` - מאגר השאלות
- `src/lib/storage.ts` - ניהול שמירה מקומית
- `src/lib/trivia.ts` - בחירת שאלה יומית
- `public/manifest.json` - הגדרות PWA
